# LDP Datastructures Hackathon of the Klausurtagung 20./21.09.2021
Groups:
* Aggregation Container
* Stack Container
* Flatten Container

## Building

To run the stack container, do:

```
$ cd stack-container
$ npm install
$ npm run build
$ npm run start 8080
```

Requires ts-node to be installed.
